# Vision Produit

## Idée

Générer automatiquement des randonnées autour de l'utilisateur en fonction d'une distance cible.

## Problème

Aujourd'hui les applications permettent surtout :
- d'enregistrer un parcours
- de suivre un parcours existant

Peu permettent de générer automatiquement une balade adaptée à un objectif simple.

## Solution

Créer une application capable de proposer plusieurs parcours pertinents autour de la position de l'utilisateur.
